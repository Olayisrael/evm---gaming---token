# EVM Gaming Token

Sample EVM-compatible gaming token contract designed for gaming and betting platforms.

## Features
- ERC-20 standard
- Mint and burn functionality
- Owner-controlled permissions
- Designed for multi-chain EVM deployment
- Frontend-ready (React / Ethers.js)

## Tech Stack
- Solidity
- OpenZeppelin

This repository represents a reference implementation. Production deployments require customization and audits.
